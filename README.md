# Raffinato ERP

Sistema de gestão interna (ERP) desenvolvido para controlar a produção artesanal de licores (Limoncello e Arancello) e xaropes.

## Funcionalidades Principais

* **Produção:** Controle completo do ciclo de vida (Maceração → Maturação → Engarrafamento).
* **Estoque de Insumos:** Gestão de matéria-prima com baixa automática e alertas de nível mínimo.
* **Estoque de Produtos:** Controle de garrafas prontas (750ml e 375ml) com separação fiscal.
* **Vendas (PDV):** Registro de pedidos, histórico por cliente e baixa automática de estoque.
* **Financeiro:** Relatórios simples de custos de insumos e receita de vendas.

## Tecnologias

* **Frontend:** Next.js 15 (App Router), React, TypeScript.
* **Estilização:** Tailwind CSS.
* **Backend/Banco de Dados:** Supabase (PostgreSQL, Auth, Edge Functions).
